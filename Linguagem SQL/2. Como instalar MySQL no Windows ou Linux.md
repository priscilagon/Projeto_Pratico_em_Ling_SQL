# Como instalar MySQL no Windows ou Linux

Instalar o MySQL no Windows ou Linux é simples e varia por SO. Use o instalador oficial no Windows ou gerenciadores de pacotes no Linux.

## Instalação no Windows

Baixe o MySQL Installer do site oficial (dev.mysql.com/downloads/installer/). Escolha "Developer Default" para incluir servidor, Workbench e ferramentas; aceite licenças, instale dependências como Visual C++ se necessário, configure senha root e rede (porta 3306 padrão).Após, teste com `mysql -u root -p` no prompt ou abra o Workbench.

## Instalação no Linux (Ubuntu/Debian)

Atualize pacotes: `sudo apt update && sudo apt upgrade`. Instale com `sudo apt install mysql-server`. Rode `sudo mysql_secure_installation` para segurança (senha root, remover anônimos). Verifique: `sudo systemctl status mysql` e acesse: `sudo mysql -u root -p`.

## Verificação e Dicas

Em ambos, inicie o serviço (Windows: Serviços; Linux: `sudo systemctl start mysql`). Para produção, configure firewall (porta 3306) e backups. Use MySQL 8.0+ para recursos modernos como JSON nativo.
